# ITS_Web – Esercizi di Sviluppo Web

Questo repository contiene esercizi sviluppati durante il corso di **Sviluppo Web** presso ITS ICT Academy.

Gli esercizi sono dedicati ai fondamenti delle tecnologie web, tra cui:

- HTML
- struttura delle pagine web
- elementi base dello sviluppo frontend

Esercizi più avanzati relativi allo stesso corso sono organizzati in repository separati dedicati a **Vite** e **React**.

Questo repository fa parte del mio percorso di apprendimento durante il programma Data Analyst presso ITS ICT Academy.
