Flex-box

Versione 1

La direzione:

  • flex-direction: column → elementi in colonna
  • flex-direction: row → default → gli elementi vanno in riga

Spaziatura orizzontale e verticale

justify-content → allinea sulla riga principale

  • justify-content: flex-start → inizio
  • justify-content: center → centro
  • justify-content: flex-end → fine
  • justify-content: space-between → spazio tra gli elementi
  • justify-content: space-around → spazio attorno

align-items → allinea sulla direzione verticale

  • align-items: flex-start;
  • align-items: center;
  • align-items: flex-end;
