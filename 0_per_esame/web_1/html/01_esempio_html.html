Punto 1.1.1. Cos’è HTML

HTML è un linguaggio di markup che definisce la struttura di una pagina web.

Una pagina HTML è divisa in head e body: nel head mettiamo le informazioni non visibili come meta, titolo e fogli di stile; nel body mettiamo il contenuto visibile all’utente.

Studiamo insieme .... scegli tu

La pagina Html è costruita da:

  • head, qui mettiamo informazioni non visibili all'utente come meta, titolo, link ai CSS e configurazioni
  • body, qui mettiamo il contenuto visibile agli utenti, testi, immagini, link, bottoni, form, liste ...

I tag principali sono:

  • titoli come h1..h6
  • testi come p e span
  • contenitori come div e section
  • link come a e attributo href
  • immagini come img con attributi src e alt
  • liste ordinate ol e non ordinate ul con li
  • form come form, input, button...

Informazioni da memorizzare ♥

Attributi globali

  1. id - identifica un elemento in modo univoco ed è utile per selezionarlo con js o css
  2. class - serve per far apartenere il contenitore div a una classe di CSS, si puo scegliere piu classi
  3. style - si usa per dare lo stile inline
  4. title - serve per mostrare info utili quando si passa il mouse sopra
  5. hidden - puo nascondere un elemento pero non rimuove dal dom
  6. data - attributi personalizzati, servono per memorizzare info nel html

Entita html

  • &lt; serve per scrivere il simbolo <
  • &gt; serve per scrivere il simbolo >
  • &amp; serve per scrivere &
  • &quot; serve per scrivere "
  • &apos; serve per '
  • &euro; serve per €
  • &rarr; serve per →
  • &larr; serve per ←
  • &hearts; serve per ♥

Struttura avanzata della paggina

  • HEADER, contiene pure NAV - collegamenti per la navigazione
  • SECTION - sezione logica della pagina
  • ASIDE - contenuto marginale es. colonna laterale, banner, pubblicita, note
  • FOOTER - copyright, contatti, link legali..
